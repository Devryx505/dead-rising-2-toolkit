# dead-rising-2-toolkit
**ULTIMATE DR2 BIG TOOL**

A complete toolkit for working with Dead Rising 2 archives and textures. It lets you open, explore, extract, replace, and rebuild BIG files while also previewing textures and file data in one place.



**BIG File Manager** – Extract BIG archives, rebuild them after editing, browse their contents without unpacking, replace individual files, process multiple archives at once, and automatically create backups before changes.

**BIG Content Viewer** – View the internal files of a BIG archive with information such as file size, type, offset, and metadata without extracting anything.

**Batch Processing** – Extract or process multiple BIG archives in sequence to speed up large modding tasks.

**File Replacement** – Replace specific files inside a BIG archive while keeping the rest of the archive intact.

**Automatic Backups** – Every modification creates a timestamped backup so the original files can always be restored.


**Hex Preview** – Inspect the raw binary data of any file inside the archive directly in a hex viewer.

**File Information Panel** – Displays detailed information including file size, type ID, offsets, and metadata.

**Search and Filtering** – Quickly locate files inside BIG archives by searching or filtering filenames.

**Archive Statistics** – Shows file counts, total size, and distribution of file types in the archive.



**Single Extract Mode** – Extract individual files from a BIG archive when you only need specific assets.

**Bulk Extract Mode** – Extract the entire archive in one operation.

**Batch Extract Mode** – Process multiple BIG files automatically.

**UnpackMany Support** – Supports Gibbed’s UnpackMany tool for high-speed batch extraction.



**Automatic Backup System** – Backups are stored in a dedicated folder with timestamps before any modification is made.

**Size Verification** – Files are validated before being replaced to avoid archive corruption.

**Error Logging** – Detailed logging system records any errors or warnings during operations.

**Recovery Options** – Files can be restored from backups if something goes wrong.



**User Interface** – Includes dark and light themes, customizable background colors, an animated title and credits, a tabbed layout for tools, a real-time operation log, and a progress bar for long operations.



**Music Integration** – Optional background music playback with looping support, volume control, and automatic detection of a music.mp3 file in the program folder.



**Texture Extraction** – Extract textures directly from archives with automatic DDS header generation and support for DXT1, DXT3, and DXT5 compression formats.

**Mipmap Support** – Preserves full mipmap chains when extracting textures.


**Texture Viewer** – Preview DDS textures with zoom controls, fit-to-view display, and detailed information about resolution, format, and size.



**File Management** – Displays a list of extracted files, allows click-to-preview viewing, supports manual DDS browsing, and automatically sanitizes filenames for compatibility.



**Gibbed Tools Integration** – Supports Gibbed.DeadRising2.Pack, Gibbed.DeadRising2.Unpack, and Gibbed.DeadRising2.UnpackMany with automatic detection when the tools are in the same directory.



**Supported Formats** – Fully supports BIG archives for reading and writing, DDS textures for extraction and preview, XML metadata parsing, and raw binary data viewing.



**Detected File Types** – Recognizes metadata files (Type 4), textures (Type 16), geometry models (Type 32), and nested BIG archives (Type 256).



**Build Features** – Distributed as a single standalone executable with no Python installation required, runs on any Windows PC, includes embedded tools and resources, and performs automatic cleanup during builds.
